

### Exidy Sorcerer 2 on Retro VGA
This is the target for Bobricius' Retro VGA board.<br/>
<img src="retrovga.png" width="300"/>

It supports the following:
* USB keyboard
* USB joysticks
* VGA video (RGB332)
* PWM sound (1 pin)
* SPI SD card

Note that currenty, the on-board keys are not functional.
